# یارِ امتحان — نسخه کامل (حساب کاربری + تاریخچه + عکس/ویدیو)

## چی اضافه شده؟
- ثبت‌نام و ورود با نام کاربری/رمز عبور
- تاریخچه چت (هر کاربر گفتگوهای قبلیش رو می‌بینه)
- ارسال عکس یا ویدیو برای تحلیل توسط هوش مصنوعی

## مرحله ۱: کلید API بگیر
برو به [aistudio.google.com](https://aistudio.google.com)، با جیمیلت وارد شو، و یه کلید رایگان بساز.

## مرحله ۲: نصب و اجرا
```bash
cd exam-helper-standalone
npm install
cp .env.example .env
```
فایل `.env` رو باز کن و:
- `GEMINI_API_KEY` رو با کلید واقعیت پر کن
- `JWT_SECRET` رو با یه رشته تصادفی طولانی و پیچیده عوض کن (این برای امنیت ورود کاربرهاست)

```bash
npm start
```
برو به `http://localhost:3000`

## نکته مهم درباره ذخیره‌سازی داده‌ها
داده‌ها (کاربرها، گفتگوها، پیام‌ها) توی یه فایل به اسم `data.json` ذخیره می‌شن. این برای شروع خوبه، ولی:
- **روی هاست‌های رایگان مثل Render**، این فایل ممکنه با هر بار Deploy یا ری‌استارت پاک بشه (چون دیسک رایگان دائمی نیست)
- اگه کاربرهات مهمن و نمی‌خوای داده‌هاشون از دست بره، باید یا:
  - از پلن پولی Render با "Persistent Disk" استفاده کنی
  - یا به یه دیتابیس واقعی و رایگان مثل MongoDB Atlas یا Supabase مهاجرت کنی (بگو تا کمکت کنم)

## نکته درباره فایل‌ها (عکس/ویدیو)
- حداکثر حجم هر فایل ۱۵ مگابایته
- فایل‌ها مستقیم به هوش مصنوعی فرستاده می‌شن و ذخیره نمی‌شن (فقط توی همون لحظه پردازش می‌شن)

## دیپلوی روی Render
همون مراحل قبلی: Build Command = `npm install`, Start Command = `npm start`, و توی Environment Variables هر دو مقدار `GEMINI_API_KEY` و `JWT_SECRET` رو اضافه کن.
